# AI뉴스자동화 v2 수정 이력 (2026-06-24)

---

## 1. fnnews RSS URL HTTPS 전환

파이낸셜뉴스 RSS 주소가 `http`에서 `https`로 301 리다이렉트되는 문제를 사전 방지하기 위해 URL을 직접 수정했습니다.

```diff
- "url": "http://www.fnnews.com/rss/r20/fn_realnews_all.xml"
+ "url": "https://www.fnnews.com/rss/r20/fn_realnews_all.xml"
```

**적용 노드:** fnnews (HTTP Request)

---

## 2. AI 요약 노드 재시도 설정

OpenAI API 호출 실패 시 자동 재시도하도록 설정을 추가했습니다.

```diff
  "name": "AI 요약",
+ "retryOnFail": true,
+ "maxTries": 3,
+ "waitBetweenTries": 1000,
```

| 설정 | 값 | 의미 |
|------|-----|------|
| `retryOnFail` | `true` | 실패 시 재시도 활성화 |
| `maxTries` | `3` | 최초 1회 + 재시도 최대 2회 |
| `waitBetweenTries` | `1000` | 재시도 간격 1초 |

**적용 노드:** AI 요약

---

## 3. HTTP Request 타임아웃 추가

RSS 서버가 응답하지 않을 때 무한 대기를 방지하기 위해 모든 HTTP Request 노드에 10초 타임아웃을 설정했습니다.

```diff
- "options": {}
+ "options": {
+   "timeout": 10000
+ }
```

**적용 노드:** YNA, Hankyung, fnnews (3개)

---

## 4. AI 노드 에러 처리 정책

AI 노드(Top3 선정, AI 요약)가 실패해도 워크플로우 전체가 중단되지 않고 다음 노드로 진행하도록 설정했습니다.

```diff
  "name": "AI 요약",
+ "onError": "continueRegularOutput",

  "name": "Top3 선정",
+ "onError": "continueRegularOutput",
```

**적용 노드:** AI 요약, Top3 선정 (2개)

## 6. 저작권 검사 자동 분기 (핵심 변경)

RSS 피드가 아닌 기사를 가져왔을 때, 기사 본문에 저작권 경고문이 있으면 AI 요약을 건너뛰고 제목만 기록하도록 자동 분기를 구현했습니다.

### 변경 전 흐름

```
Parse Top 3 → AI 요약 → Merge Summary → Prepare For Notion → Save to Notion
```

### 변경 후 흐름

```
Parse Top 3
    │
    ▼
기사 원문 가져오기 (HTTP Request, 10초 타임아웃)
    │
    ▼
저작권 검사 (Code)
    │
    ▼
저작권 분기 (IF: hasCopyright?)
    │
    ├── true (저작권 경고 발견)
    │       └→ 제목만 사용 (Code) → Prepare For Notion
    │
    └── false (정상)
            └→ AI 요약 (OpenAI) → Prepare For Notion
                                        │
                                        ▼
                                  Save to Notion
```

### 추가된 노드 (4개)

| 노드 | 타입 | 역할 |
|------|------|------|
| **기사 원문 가져오기** | HTTP Request | 기사 링크의 HTML 원문 수집 (타임아웃 10초) |
| **저작권 검사** | Code | 본문에서 저작권 키워드 감지 |
| **저작권 분기** | IF | `hasCopyright` 값에 따라 분기 |
| **제목만 사용** | Code | 경고 메시지와 함께 제목만 출력 |

### 저작권 감지 키워드

```
무단전재, 재배포 금지, 무단 복제, 무단복제,
저작권자, 무단 전재, 기사 원문의 저작권,
무단전재 및 재배포, 저작권법에 의해
```

### 저작권 기사 출력 예시

저작권 경고가 감지된 기사는 Notion에 아래와 같이 저장됩니다:

| 필드 | 값 |
|------|-----|
| 제목 | 원래 기사 제목 |
| 요약문 | ⚠️ 저작권 보호 기사 - AI 요약 불가. 제목만 기록합니다. |
| 원문링크 | 기사 링크 |
| 언론사 | 원래 언론사명 |

### 제거된 노드

| 노드 | 이유 |
|------|------|
| **Merge Summary** | IF 분기로 인해 위치 기반 병합(combineByPosition)이 불가능해져 제거. Prepare For Notion에서 직접 데이터를 처리하도록 변경 |

### 수정된 노드

| 노드 | 변경 내용 |
|------|----------|
| **Prepare For Notion** | `summary` 우선순위 변경 — `item.json.summary`를 `item.json.output`보다 먼저 확인하여 "제목만 사용" 경로의 요약문이 정상 반영되도록 수정 |
| **Save to Notion** | 노드 위치만 오른쪽으로 이동 (새 노드 공간 확보) |
